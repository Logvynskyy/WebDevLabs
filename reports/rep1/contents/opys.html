Нашою бригадою було проаналізовано багато різних тем, але ми зупинилися на темі під назвою "Car shop". Адже на нашому ринку існує лише один сайт для покупки авто. Реалізація цієї теми передбачає реалізацію веб-застосунку з функціоналом продажу та демонстрації автомобілів. Буде додано функціонал адміністратора та звичайного клієнта. У адміністратора будуть можливості створювати, редагувати та видаляти оголошення. Також він буде мати змогу формувати звіти та отримувати детальну статистику з продажів на веб-сайту. Користувач буде мати змогу зареєструватися на сайті та мати змогу розмістити оголшення з продажу авто.Також корістувач буде мати змогу залишити за'явку на сайті щодо автомобілів які його цікавлять.Додамо функціонал поривняння та пошуку автомобілів за певними характеристиками, що дасть змогу користувачам обрати для себе авто з найкращими характеристиками з поміж представлених.

Функціональні вимоги

  • Користувач має можливість переглянути таблиці з назвами марок авто, та має можливість ввести своє ім'я у спеціальну форму.
  • Сайт повинен мати поля для пошуку авто за параметрами
  • Сайт повинен мати фільтри для сортування оголошень
  • Сайт повинен мати систему відгуків
  • Сайт повинен мати сторінки з інформацією про роботу автомобілів
